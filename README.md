# 🌦️ Weather Forecast Web Application

A dynamic and interactive weather forecasting web application that provides real-time weather updates, location-based forecasts, and detailed hourly insights with a user-friendly interface.

## 🚀 Features

- 🔍 **Smart Location Search**
  - Auto-suggestion for locations while typing
  - Quickly find cities with partial input (e.g., "ch" → Chennai, Chicago)

- 🗺️ **Map-Based Location Selection**
  - Click on the map to select any location
  - Instantly fetch weather data for selected coordinates

- 📍 **Current Location Detection**
  - Get weather details using your device location

- 📅 **7-Day Weather Forecast**
  - Displays daily weather conditions for the upcoming week

- ⏱️ **Hourly Weather Details**
  - Click on a forecast card to view detailed hourly data

- 🎥 **Dynamic Visual Experience**
  - Weather-based background videos and images (clear, rain, storm, etc.)

- ⚡ **Real-Time Data Fetching**
  - Integrated with external weather API for live updates
